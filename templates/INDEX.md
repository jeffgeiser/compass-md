## How to use this index

Agents reading a large Compass should:
1. Read `COMPASS.md` first for conventions
2. Read this `INDEX.md` to find relevant files
3. Read only the files relevant to the current task

Update this file when adding new perspective or criteria files. The one-line summaries save agents from opening files that aren't relevant.
